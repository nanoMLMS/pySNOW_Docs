# pySNOW tutorials

We have developed a collection of tutorials which you should get together with the code when you clone the pySNOW github folder.
These cover many use cases, tips and tricks for the functions in the library.
You can find them in the `tutorial/` folder on the github repo, or directly from here:

### Input/Output
- [I/O and rototranslations](https://github.com/nanoMLMS/pySNOW/blob/master/tutorial/io_rotations.ipynb)

### Analysis tools
- [Distributions](https://github.com/nanoMLMS/pySNOW/blob/master/tutorial/distributions.ipynb)
- [Bimetallic distributions](https://github.com/nanoMLMS/pySNOW/blob/master/tutorial/distributions_bimetallic.ipynb)
- [(Generalized) coordination numbers](https://github.com/nanoMLMS/pySNOW/blob/master/tutorial/coordination.ipynb)
- [Common neighbour analysis](https://github.com/nanoMLMS/pySNOW/blob/master/tutorial/cna.ipynb)
- [Mean squared displacements](https://github.com/nanoMLMS/pySNOW/blob/master/tutorial/msd.ipynb)
- [SAXS spectra](https://github.com/nanoMLMS/pySNOW/blob/master/tutorial/saxs.ipynb)
- [Shape descriptors](https://github.com/nanoMLMS/pySNOW/blob/master/tutorial/shape.ipynb)

### System manipulation
- [Adding adsorbed atoms and molecules to your system](https://github.com/nanoMLMS/pySNOW/blob/master/tutorial/add_molecule.ipynb)
